# -password-manager-python
CLI-based password manager using Python and file handling


##  Description
A simple CLI-based password manager built using Python. It allows users to store and view passwords securely using file handling.

##  Features
- Add new passwords
- View saved passwords
- File-based storage
- Simple menu-driven interface

##  Technologies Used
- Python
- File Handling
- Exception Handling

##  How to Run
1. Download the file
2. Run: python filename.py
