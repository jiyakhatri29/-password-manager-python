# password-manager-python
CLI-based password manager using Python and file handling


##  Description
This is a simple command-line based Password Manager built using Python.
It allows users to securely store and retrieve login credentials using file handling.
##  Features
Add new website credentials (website, username, password)
View all saved passwords
Persistent storage using text file
Exception handling for file operations
Menu-driven user interface

##  Technologies Used
Python
File Handling
Exception Handling
CLI (Command Line Interface)

##  Project Structure

password-manager-python/
│── main.py
│── password.txt
│── README.md

##  How to Run
Clone the repository:
git clone https://github.com/jiyakhatri29/-password-manager-python.git
Navigate to the folder:
cd -password-manager-python
Run the program:
python main.py
